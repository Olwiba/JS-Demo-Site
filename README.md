## How to get started?
1. Clone the repository  
`git clone https://github.com/Olwiba/JS-Demo-Site.git`
2. Install dependencies (express server)  
`npm install` or `yarn install`
3. Start the server  
`npm start` or `yarn start`
4. Open the website  
http://localhost:3000/

Happy coding!
